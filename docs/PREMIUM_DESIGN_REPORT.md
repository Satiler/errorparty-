## 🎨 ДИЗАЙН-СИСТЕМА

**Цвета:**
```css
Фон страницы:     linear-gradient(to bottom, #1a1a1a, #000, #000)
Карточки:         #171717/40 с backdrop-blur
Hover карточек:   #292929/60
Акцент Play:      linear-gradient(135deg, #4ade80, #22c55e)
Текст основной:   #ffffff
Текст вторичный:  #9ca3af
Hover текст:      #4ade80 (зеленый)
```

**Размеры:**
```
Заголовок H1:        text-8xl (96px)
Featured карточки:   w-32 h-32 иконка (128px)
Обычные карточки:    text-7xl иконка (72px)
Обложка плейлиста:   w-72 h-72 (288px)
Play кнопка главная: w-20 h-20 (80px)
Play кнопка карточки: w-14 h-14 (56px)
```

**Анимации:**
```javascript
// Появление элементов
initial={{ opacity: 0, y: 20 }}
animate={{ opacity: 1, y: 0 }}
transition={{ delay: index * 0.05 }} // stagger

// Hover на карточках
whileHover={{ y: -8 }} // подъем на 8px
transition={{ duration: 0.2 }}

// Play кнопка
whileHover={{ scale: 1.1 }}
whileTap={{ scale: 0.95 }}

// Градиентный блеск
<div className="absolute inset-0 bg-gradient-to-r from-transparent via-white/5 to-transparent 
  opacity-0 group-hover:opacity-100 transition-opacity duration-500 
  transform -skew-x-12 group-hover:translate-x-full" />
```

**Тени:**
```css
Карточки:       hover:shadow-2xl hover:shadow-purple-500/20
Play кнопка:    shadow-2xl shadow-green-500/50
Обложка:        shadow-2xl с динамическим цветом плейлиста
```
